# B. Задача о рюкзаке с восстановлением ответа

## Описание задачи

Есть `n` предметов. Для каждого предмета известны:

- вес `p[i]`;
- стоимость `c[i]`.

Есть рюкзак грузоподъёмности `w`. Нужно выбрать набор предметов так, чтобы:

- суммарный вес не превышал `w`;
- суммарная стоимость была максимальной.

Кроме значения оптимального ответа, нужно восстановить один подходящий набор предметов.

## Идея решения

Это классическая задача о рюкзаке `0/1`, где каждый предмет можно взять не более одного раза.

Обозначим:

- `dp[i][j]` — максимальная стоимость, которую можно получить, рассматривая первые `i` предметов и имея ограничение по весу `j`.

Тогда для предмета `i` есть два варианта:

- не брать его;
- взять его, если его вес не превышает текущую вместимость.

Переход:

- если `p[i - 1] > j`, то `dp[i][j] = dp[i - 1][j]`;
- иначе `dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - p[i - 1]] + c[i - 1])`.

## Почему это работает

Для каждого состояния `(i, j)` оптимальное решение либо использует `i`-й предмет, либо нет.

Других вариантов нет.

Если предмет не берётся, ответ совпадает с `dp[i - 1][j]`.  
Если берётся, остаётся оптимально добрать решение на вес `j - p[i - 1]` из первых `i - 1` предметов.

Максимум из этих двух вариантов и есть оптимальный ответ.

## Восстановление ответа

После заполнения таблицы начинаем идти из состояния `(n, w)` назад:

- если значение пришло сверху, предмет не брали;
- иначе предмет был взят, и тогда:
  - добавляем его в ответ;
  - уменьшаем текущий вес на его массу.

Так можно восстановить один из оптимальных наборов предметов.

## Алгоритм

1. Считать веса и стоимости.
2. Построить таблицу `dp` размера `(n + 1) x (w + 1)`.
3. Заполнить её по стандартному переходу задачи о рюкзаке.
4. Из `dp[n][w]` восстановить выбранные предметы обратным проходом.
5. Вывести суммарную стоимость, суммарный вес, число предметов и сами веса/стоимости выбранных предметов.

## Сложность

- время: `O(nw)`;
- память: `O(nw)`.

## Итог

Задача решается стандартной двумерной динамикой по предметам и допустимому весу, а сам набор предметов восстанавливается обратным проходом по таблице.
