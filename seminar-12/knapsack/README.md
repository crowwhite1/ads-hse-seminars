# Задача о рюкзаке

Задача о рюкзаке — это классическая задача динамического программирования.

Есть:

- набор предметов;
- вес каждого предмета;
- стоимость каждого предмета;
- максимальная вместимость рюкзака.

Нужно выбрать подмножество предметов с максимальной суммарной стоимостью так, чтобы суммарный вес не превышал ограничение.

## Вариант задачи

Здесь рассматривается вариант `0/1`:

- каждый предмет можно либо взять один раз,
- либо не брать совсем.

## Идея алгоритма

Пусть:

- `dp[i][j]` — максимальная стоимость, которую можно получить, если рассматривать первые `i` предметов и иметь ограничение по весу `j`.

Для `i`-го предмета есть два варианта:

- не брать его;
- взять его, если он помещается.

## Переход

Если вес предмета `weights[i - 1]` больше `j`, то взять его нельзя:

`dp[i][j] = dp[i - 1][j]`

Иначе:

`dp[i][j] = max(dp[i - 1][j], dp[i - 1][j - weights[i - 1]] + costs[i - 1])`

## Базовые случаи

- `dp[0][j] = 0` для всех `j`;
- `dp[i][0] = 0` для всех `i`.

Если нет предметов или нет допустимого веса, максимальная стоимость равна нулю.

## Восстановление ответа

После заполнения таблицы можно пройти из состояния `(n, capacity)` назад:

- если значение пришло сверху, предмет не брали;
- иначе предмет был выбран.

Так можно восстановить один из оптимальных наборов предметов.

## Наглядный пример

Пусть:

- вместимость рюкзака `w = 5`;
- веса: `[2, 3, 4]`;
- стоимости: `[6, 7, 10]`.

Тогда возможны такие полезные варианты:

- взять предмет 1: вес `2`, стоимость `6`;
- взять предмет 2: вес `3`, стоимость `7`;
- взять предмет 3: вес `4`, стоимость `10`;
- взять предметы 1 и 2: вес `5`, стоимость `13`.

Предметы 1 и 3 вместе уже не помещаются, потому что их общий вес `6`.

Значит оптимальный ответ:

- суммарный вес `5`;
- суммарная стоимость `13`;
- выбранные предметы: 1 и 2.

Именно такой ответ и должна восстановить динамика.

## Почему это работает

В любом оптимальном решении для состояния `(i, j)` `i`-й предмет либо участвует, либо нет.

Если он не участвует, остаётся задача на первых `i - 1` предметах.

Если участвует, нужно добавить его стоимость к оптимальному ответу для меньшей вместимости `j - weight`.

Перебор этих двух вариантов покрывает все возможные решения, а максимум даёт оптимальный ответ.

## Сложность

Пусть:

- `n` — число предметов;
- `w` — вместимость рюкзака.

Тогда:

- время: `O(nw)`;
- память: `O(nw)`.

## Итог

Задача о рюкзаке — базовый пример динамики, где состояние задаётся числом рассмотренных предметов и текущим ограничением по весу.
