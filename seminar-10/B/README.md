# B. Построение

## Описание задачи

Дано `N` солдат и `M` утверждений вида:

- солдат `A` выше солдата `B`.

Нужно определить, можно ли построить солдат в один ряд так, чтобы все эти утверждения выполнялись одновременно.

Если можно, требуется вывести один подходящий порядок.

## Сведение к графу

Построим ориентированный граф:

- вершины — солдаты;
- ребро `A -> B` означает, что `A` должен стоять раньше `B`, потому что он выше.

Тогда задача превращается в поиск топологического порядка вершин.

## Когда ответ существует

Топологическая сортировка существует тогда и только тогда, когда в графе нет циклов.

Если есть цикл, получается противоречие вида:

- `A` выше `B`,
- `B` выше `C`,
- `C` выше `A`.

Такой набор условий выполнить невозможно.

## Идея решения

Используется DFS с раскраской вершин:

- `0` — вершина не посещена;
- `1` — вершина находится в стеке рекурсии;
- `2` — вершина полностью обработана.

Во время обхода:

- если переходим в вершину цвета `0`, продолжаем DFS;
- если видим ребро в вершину цвета `1`, найден цикл;
- после обработки всех исходящих рёбер добавляем вершину в список ответа.

После завершения обхода список разворачивается.

## Почему это работает

Если граф ацикличен, то DFS-порядок выхода даёт топологическую сортировку:

- вершина добавляется в ответ только после всех вершин, в которые из неё есть путь по обязательным зависимостям;
- после разворота каждая вершина оказывается раньше всех, кто должен стоять после неё.

Если в графе найдено обратное ребро в вершину цвета `1`, это означает цикл.  
Тогда никакого допустимого построения не существует.

## Алгоритм

1. Построить ориентированный граф по ограничениям.
2. Запустить DFS из всех непосещённых вершин.
3. Одновременно:
   - проверять наличие цикла;
   - строить порядок по времени выхода.
4. Если найден цикл, вывести `No`.
5. Иначе развернуть порядок и вывести `Yes` и найденную перестановку.

## Сложность

Пусть:

- `N` — число солдат;
- `M` — число ограничений.

Тогда:

- время: `O(N + M)`;
- память: `O(N + M)`.

## Итог

Задача является классической проверкой существования топологической сортировки.

Если граф ограничений ацикличен, найденный топологический порядок и будет корректным построением солдат.
