# Algorithms and Data Structures — Seminars

Данный репозиторий содержит примеры кода и решения задач по курсу  
**«Алгоритмы и структуры данных»**, читаемому в **НИУ ВШЭ**  
на образовательной программе **«Бизнес-информатика»**.

Материалы относятся к **2025–2026 учебному году**  
(начиная с 2026 года).

---

## О репозитории

В репозитории сохраняется код, реализованный и разобранный
после семинарских занятий по курсу.

Основное назначение репозитория:
- систематизация материалов семинаров;
- хранение реализаций алгоритмов;
- повторение и самостоятельная практика;
- подготовка к контрольным, экзамену и собеседованиям.

Код добавляется и обновляется по мере прохождения курса.

---

## Используемые языки программирования

В рамках курса задачи реализуются на нескольких языках:

- Python  
- Java  
- C++

---

## Структура репозитория

Репозиторий организован **по семинарам**.

Каждый семинар находится в отдельной директории `seminar-XX`.
Внутри семинара задачи разбиты по буквам (`A`, `B`, `C`, …),
а внутри каждой задачи — по языкам программирования.

### Пример структуры:

seminar-01/
├── A/
│ ├── python/
│ │ └── A.py
│ ├── java/
│ │ └── A.java
│ ├── cpp/
│ │ └── A.cpp
│ └── README.md
├── B/
├── C/
├── D/
├── E/
├── F/
└── G/

- `seminar-01` — номер семинара  
- `A`, `B`, `C`, … — задачи семинара  
- `python`, `java`, `cpp` — реализации на соответствующих языках  

---

## README внутри задач

Для задач внутри папки задачи присутствует `README.md`,
в котором кратко описаны:
- условие задачи;
- идея решения;
- используемые подходы и оценки сложности.

---

## Формат кода

- каждая программа решает одну конкретную задачу;
- код ориентирован на учебное использование и читаемость;
- возможны несколько реализаций одной задачи (разные подходы).
