# Объединение последовательностей квадратов и кубов

## Условие
Даны две бесконечные возрастающие последовательности:

- `A`: `A_i = i²`
- `B`: `B_i = i³`

Необходимо построить возрастающую последовательность `C`,
полученную объединением `A` и `B`.

Если некоторое число встречается **и в `A`, и в `B`**,  
то в `C` оно добавляется **один раз**.

По заданному `x` требуется найти элемент `C_x`.

---

## Идея решения

Задача решается **двумя указателями**, аналогично этапу `merge`
в сортировке слиянием:

- указатель `a` — для квадратов
- указатель `b` — для кубов

На каждом шаге:
1. Сравниваем `a²` и `b³`
2. В `C` добавляем меньшее
3. Если значения равны — добавляем один раз и двигаем оба указателя

Повторяем `x` раз — последний добавленный элемент и есть `C_x`.

---

## Почему это эффективно

- Не строим массив `C`
- Не храним последовательности
- Работаем строго по формуле

---

## Сложность

- **Время:** `O(x)`
- **Память:** `O(1)`

---

## Замечания

- Используется тип `long long / long`, так как значения могут быть большими в Java/Cpp
- Решение работает для `x` до `10⁷` без проблем по памяти
- Это классический пример **слияния отсортированных последовательностей**

---

## Пример

Ввод:
2

Ход:
A: 1, 4, 9, ...
B: 1, 8, 27, ...
C: 1, 4, 8, 9, ...

Вывод:
4
