# B — Сортировка слиянием (Merge Sort)

## Идея
Используется сортировка слиянием:

1) Рекурсивно делим массив пополам, пока не останутся отрезки длины 0/1  
2) Сливаем два отсортированных подмассива в один, выбирая меньший из “голов”  
3) Повторяем вверх по рекурсии

Чтобы ускорить и не плодить списки, используется **один общий буфер** `buf`.

## Стабильность
Сортировка **стабильная**: при равных значениях берём элемент из левой части (`<=`),
поэтому относительный порядок равных элементов сохраняется.

## Сложность
- Время: `O(N log N)` (всегда)
- Память: `O(N)` на буфер

## Ввод/вывод
- Ввод: `N`, затем `N` целых чисел
- Вывод: числа в порядке неубывания
