# C — Быстрая сортировка (Quick Sort)

## Идея
Используется быстрая сортировка с **трёхпутевым разбиением** массива
относительно опорного элемента (`pivot`):

- `< pivot`
- `== pivot`
- `> pivot`

Это позволяет эффективно работать с массивами, содержащими
много одинаковых элементов.

Опорный элемент выбирается **детерминированно**:
    `pivot = a[(l + r) / 2]`


---

## Алгоритм
1. Если длина отрезка меньше 2 — выход
2. Проходим по массиву и перераспределяем элементы в три группы
3. Рекурсивно сортируем части `< pivot` и `> pivot`

---

## Стабильность
❌ **Нестабильная сортировка**  
(относительный порядок равных элементов может измениться)

---

## Сложность

### Время
- Лучший случай: `O(N log N)`
- Средний случай: `O(N log N)`
- Худший случай: `O(N²)`  
  (в данной реализации вероятность сильно снижена за счёт трёхпутевого разбиения)

### Память
- `O(log N)` — глубина рекурсии

---

## Почему используется трёхпутевое разбиение
- Существенно ускоряет работу при большом количестве одинаковых элементов
- Уменьшает глубину рекурсии
- Часто используется в промышленной реализации Quick Sort

---

## Ввод / вывод
- Ввод: `N`, затем `N` целых чисел
- Вывод: массив в порядке неубывания
