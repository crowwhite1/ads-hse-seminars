# D — Количество различных марок

## Задача
Дан массив из `N` чисел (идентификаторы марок). Нужно вывести количество **различных** значений.

⚠️ В примечании указано, что решение должно использовать **сортировку** (решения через `set` не принимаются).

---

## Идея решения
1. Отсортировать массив по неубыванию.
2. Пройти по массиву и посчитать, сколько раз значение меняется по сравнению с предыдущим элементом.

После сортировки все одинаковые элементы стоят подряд, поэтому подсчёт делается одним проходом.

---

## Сложность
- Сортировка: `O(N log N)`
- Подсчёт уникальных: `O(N)`
- Память: `O(1)` дополнительной (кроме хранения массива)

---

## Формат ввода
- Первая строка: `N` (`1 ≤ N ≤ 100000`)
- Вторая строка: `N` целых чисел

## Формат вывода
- Одно число — количество различных значений
