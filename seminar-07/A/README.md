### Heap: Insert / Extract Max

**Задача:** реализовать структуру данных `Heap` для целых чисел, поддерживающую операции:

1. `Insert(k)` — добавить число `k`
2. `Extract` — извлечь **наибольшее** число (и удалить его)

Гарантируется, что при выполнении `Extract` в куче есть хотя бы один элемент.

### Формат ввода

* В первой строке число команд `N` (`1 ≤ N ≤ 100000`)
* Далее `N` строк команд:

  * `0 x` — вставить `x` (`1 ≤ x ≤ 1000000`)
  * `1` — извлечь максимум

### Формат вывода

Для каждой команды `1` вывести извлечённое число в отдельной строке.

### Идея решения

Используется **двоичная максимальная куча (binary max-heap)** на массиве.

* `push(x)`:

  * добавляем элемент в конец
  * поднимаем вверх (`shift_up`), пока не восстановится свойство кучи
* `pop()`:

  * забираем вершину (максимум)
  * переносим последний элемент в корень
  * опускаем вниз (`shift_down`), восстанавливая свойство кучи

### Сложность

* `Insert` — `O(log n)`
* `Extract` — `O(log n)`
* Память — `O(n)`
