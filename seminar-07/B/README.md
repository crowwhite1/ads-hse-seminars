### Heap Sort (пирамидальная сортировка)

**Задача:** отсортировать массив из `N` целых чисел по неубыванию, используя пирамидальную сортировку.


### Формат ввода

* В первой строке: `N` (`N ≤ 10^5`)
* Во второй строке: `N` целых чисел, по модулю не больше `10^9`

### Формат вывода

Вывести числа в порядке неубывания.


### Идея решения

Используем **минимальную двоичную кучу (MinHeap)**:

1. Загружаем все элементы массива в кучу и делаем `heapify()` за `O(N)`
2. `N` раз извлекаем минимум (`pop()`), получая отсортированную последовательность

Так как каждый `pop()` стоит `O(log N)`, итоговая сложность:

* `heapify`: `O(N)`
* `N` извлечений: `O(N log N)`
* память: `O(N)`
