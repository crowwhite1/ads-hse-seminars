### Дейкстра из вершины 0 (ориентированный взвешенный граф)

**Задача:** найти длины кратчайших путей из вершины `0` до всех остальных вершин в ориентированном взвешенном графе.


### Ввод

* `n m` — число вершин и дуг
* далее `m` строк: `u v w` — дуга `u -> v` веса `w`
  Гарантируется, что из `0` существует путь до всех вершин.

Ограничения (по условию):

* `2 ≤ n ≤ 1e5`
* `n-1 ≤ m ≤ min(n(n-1), 2e5)`
* `1 ≤ w ≤ 1e9`


### Вывод

Для каждой вершины `i = 1..n-1` вывести `dist[i]` (кратчайшее расстояние от `0` до `i`) в отдельной строке.


### Алгоритм

Используется алгоритм Дейкстры с минимальной кучей.
В куче хранятся пары `(расстояние, вершина)`.


### Сложность

* Время: `O((n + m) log m)`
* Память: `O(n + m)`
