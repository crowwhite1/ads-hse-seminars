# Задача E. Кафешки

## 📌 Условие

Дан неориентированный связный граф:
- вершины — факультеты
- рёбра — дороги с длинами

Есть главный факультет `C`.

Кафешка находится в любой точке (вершина или точка на ребре), для которой:

👉 кратчайшее расстояние до `C` равно ровно `len`


## 🎯 Задача

Найти:
> сколько таких точек (вершин и точек на рёбрах) существует


## 🔢 Формат входных данных

- `A` — число вершин (до `10^4`)
- `B` — число рёбер
- `C` — стартовая вершина  
- далее список рёбер `(u, v, w)`
- число `len`


## 📤 Формат выходных данных

Вывести одно число — количество точек на расстоянии `len`


## 💡 Идея задачи

Это задача на:
> **алгоритм Дейкстры + геометрия на рёбрах**


## 🧠 Ключевая идея

### 🔹 Шаг 1. Считаем расстояния

Запускаем **Дейкстру от вершины `C`**  
→ получаем `dist[v]` — кратчайшее расстояние до всех вершин


### 🔹 Шаг 2. Считаем вершины

Если:
```

dist[v] == len

```
👉 эта вершина подходит


### 🔹 Шаг 3. Считаем точки на рёбрах

Рассматриваем каждое ребро `(u, v, w)`

Идея:
> возможно, точка на ребре лежит на расстоянии `len`


## ⚙️ Разбор случаев

Пусть:
```

du = dist[u]
dv = dist[v]

```


### 🔹 Случай 1: обе вершины ближе

```

du < len и dv < len

```

Проверяем:
```

du + dv + w

```

- если `== 2 * len` → 1 точка
- если `> 2 * len` → 2 точки


### 🔹 Случай 2: одна сторона "пересекает len"

```

du < len < dv   или   dv < len < du

```

👉 ровно 1 точка


### 🔹 Случай 3: одна вершина ровно на len

```

du == len

```

Проверяем:
```

dv < len и dv + w > du

```

👉 тогда +1 точка

(аналогично для `v`)


## 🧩 Почему это работает

Мы ищем:
> точки, где кратчайший путь = len

Это может быть:
- либо вершина
- либо точка внутри ребра


## ⏱ Сложность

- Дейкстра: **O((A + B) log A)**
- обработка рёбер: **O(B)**

👉 итог: **O(B log A)**


## 📘 Важные моменты

- граф неориентированный
- нельзя считать вершины дважды
- аккуратно обрабатывать равенства
- использовать `long long` / `int64`


## 🎯 Зачем эта задача

- практика Дейкстры
- работа с расстояниями
- нестандартная обработка рёбер
