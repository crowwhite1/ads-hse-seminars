# D. Обход графа — размер компоненты связности

Дан **неориентированный невзвешенный граф**, заданный **матрицей смежности** `N × N`.
Нужно найти **количество вершин в компоненте связности**, содержащей заданную вершину `S`
(включая саму `S`).

---

## Входные данные

- В первой строке: `N` и `S`  
  `1 ≤ N ≤ 100`, `1 ≤ S ≤ N`
- Далее `N` строк по `N` чисел `0/1` — матрица смежности.
- На главной диагонали стоят нули.

---

## Выходные данные

Одно число — размер компоненты связности вершины `S`.

---

## Идея решения

1. Преобразуем матрицу смежности в список смежности:
   - если `matrix[i][j] = 1`, добавляем ребро `i -> j`
2. Запускаем **DFS** из вершины `S`.
3. Считаем, сколько вершин помечено `visited = true`.

---

## Корректность

DFS посещает ровно те вершины, которые достижимы из `S` по рёбрам графа.  
В неориентированном графе это означает, что мы посещаем **всю компоненту связности** вершины `S`.
Количество посещённых вершин и есть ответ.

---

## Сложность

- Построение списка смежности из матрицы: `O(N^2)`
- DFS по списку смежности: `O(N + E)`  
  (но при чтении матрицы всё равно доминирует `O(N^2)`)

Память:
- `O(N + E)` на список смежности и массив `visited`.
