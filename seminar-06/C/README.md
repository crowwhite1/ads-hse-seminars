# C. Истоки и стоки (ориентированный граф)

## Определения
- **Исток** — вершина, в которую **не входит ни одно ребро** (in-degree = 0).
- **Сток** — вершина, из которой **не выходит ни одно ребро** (out-degree = 0).

Граф задан **матрицей смежности** `n × n` (0/1), ориентированный.

---

## Идея решения
Мы читаем матрицу построчно.

### Как найти стоки
Для вершины `i` смотрим `i`-ю строку:
- если в строке нет ни одной `1`, значит у `i` нет исходящих ребер ⇒ `i` — **сток**.

### Как найти истоки
Заведём массив `isSource[j] = true` для всех `j`.
Когда читаем элемент матрицы `a[i][j]`:
- если `a[i][j] == 1`, то в вершину `j` **входит** ребро ⇒ `isSource[j] = false`.

После чтения всей матрицы:
- все `j`, у которых `isSource[j] == true`, являются **истоками**.

---

## Формат вывода
Как в твоём решении на Python:
1) сначала `k` — количество истоков  
2) затем `k` строк с номерами истоков (по одному в строке)  
3) затем `t` — количество стоков  
4) затем `t` строк с номерами стоков  

Нумерация вершин: **с 1**.

---

## Сложность
- Время: `O(n^2)` — нужно прочитать всю матрицу.
- Память: `O(n)` — храним только `isSource`, `sources`, `sinks`.

`n ≤ 3000`, поэтому важно читать быстро (в Java `split()` обычно слишком медленный на 9e6 чисел).
