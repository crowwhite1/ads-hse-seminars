# Матрица смежности → список смежности (ориентированный граф)

## Условие
Дан ориентированный невзвешенный граф (до 100 вершин) в виде матрицы смежности `n × n`,
где `1` означает наличие ребра `i -> j`, а `0` — отсутствие.

Нужно вывести список смежности:
для каждой вершины `i` в отдельной строке вывести:
- `m` — количество исходящих рёбер из `i`,
- затем `m` номеров вершин, в которые есть ребро из `i`,
в порядке возрастания номеров.

Нумерация вершин в ответе — **с 1**.

---

## Идея решения
Для каждой строки `i` матрицы:
1. считаем количество единиц (это `m`);
2. если `m > 0`, второй проход по строке — выписываем все `j+1`, где `arr[i][j] == 1`.

Порядок по возрастанию обеспечивается автоматически, потому что `j` перебирается слева направо.

---

## Корректность
- Каждая единица в строке `i` соответствует ребру `i -> j`, значит вершина `j` должна быть в списке смежности `i`.
- Мы выводим ровно все такие `j`, поэтому список смежности верный.
- `j` перебирается по возрастанию, значит вывод отсортирован.

---

## Сложность
- Время: `O(n^2)` (матрица `n × n`, два прохода по каждой строке — это всё равно `O(n^2)`).
- Память: `O(n^2)` на хранение матрицы.

При `n ≤ 100` это очень легко проходит по ограничениям.
