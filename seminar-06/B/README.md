# B. Дерево (проверка по матрице смежности)

## Условие
Дан неориентированный граф без петель и кратных рёбер в виде матрицы смежности `N × N` (`1` — ребро есть, `0` — нет).
Матрица симметрична относительно главной диагонали.

Нужно вывести:
- `YES`, если граф является деревом
- `NO` иначе

`1 ≤ N ≤ 100`

---

## Когда граф — дерево
Неориентированный граф является деревом **тогда и только тогда**, когда выполняются два условия:
1) граф **связный**
2) число рёбер **E = N - 1**

(Эквивалентно: связность + отсутствие циклов, но через `E = N-1` удобнее.)

---

## Идея решения
1) Читаем матрицу и считаем число рёбер `E`, но чтобы не посчитать одно ребро дважды,
   учитываем только элементы выше диагонали: `j > i`.
2) Запускаем DFS из вершины `0` и проверяем, что все вершины посещены → граф связный.
3) Если `E == N - 1` и граф связный → `YES`, иначе `NO`.

---

## Почему это корректно
- Если граф дерево, то он связный и содержит ровно `N-1` ребро → алгоритм выдаст `YES`.
- Если алгоритм выдал `YES`, значит граф связный и `E = N-1`.
  Связный неориентированный граф с `N-1` ребром не содержит циклов, значит это дерево.

---

## Сложность
- Чтение матрицы: `O(N^2)`
- DFS по матрице (проверяем всех соседей через строку матрицы): `O(N^2)`
- Итог: `O(N^2)` по времени, `O(N^2)` по памяти (если хранить матрицу)

При `N ≤ 100` это полностью нормально.
