# Deque (–¥–≤—É—Å—Ç–æ—Ä–æ–Ω–Ω—è—è –æ—á–µ—Ä–µ–¥—å) ‚Äî —Å—Ç—Ä—É–∫—Ç—É—Ä–∞ –¥–∞–Ω–Ω—ã—Ö

–ì–ª–∞–≤–Ω–∞—è –∏–¥–µ—è:  
‚úÖ **push/pop** –≤–æ–∑–º–æ–∂–Ω—ã **–∏ —Å–ª–µ–≤–∞, –∏ —Å–ø—Ä–∞–≤–∞**.


## –ö–∞–∫ —ç—Ç–æ –æ—Ç–ª–∏—á–∞–µ—Ç—Å—è –æ—Ç Queue –∏ Stack

- **Queue (FIFO)**: –¥–æ–±–∞–≤–ª—è–µ–º –≤ –∫–æ–Ω–µ—Ü, —É–¥–∞–ª—è–µ–º –∏–∑ –Ω–∞—á–∞–ª–∞  
- **Stack (LIFO)**: –¥–æ–±–∞–≤–ª—è–µ–º –∏ —É–¥–∞–ª—è–µ–º —Å –æ–¥–Ω–æ–π —Å—Ç–æ—Ä–æ–Ω—ã (–≤–µ—Ä—à–∏–Ω–∞)  
- **Deque**: –¥–æ–±–∞–≤–ª—è–µ–º –∏ —É–¥–∞–ª—è–µ–º **—Å –æ–±–µ–∏—Ö —Å—Ç–æ—Ä–æ–Ω**


## –û—Å–Ω–æ–≤–Ω—ã–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ Deque

–û–±—ã—á–Ω–æ –ø–æ–¥–¥–µ—Ä–∂–∏–≤–∞—é—Ç—Å—è 8 –æ–ø–µ—Ä–∞—Ü–∏–π (4 –º–æ–¥–∏—Ñ–∏—Ü–∏—Ä—É—é—â–∏–µ + 4 –ø—Ä–æ–≤–µ—Ä–∫–∏):

### –î–æ–±–∞–≤–ª–µ–Ω–∏–µ
- **push_front(x)** ‚Äî –¥–æ–±–∞–≤–∏—Ç—å `x` –≤ –Ω–∞—á–∞–ª–æ (—Å–ª–µ–≤–∞)
- **push_back(x)** ‚Äî –¥–æ–±–∞–≤–∏—Ç—å `x` –≤ –∫–æ–Ω–µ—Ü (—Å–ø—Ä–∞–≤–∞)

### –£–¥–∞–ª–µ–Ω–∏–µ
- **pop_front()** ‚Äî —É–¥–∞–ª–∏—Ç—å –∏ –≤–µ—Ä–Ω—É—Ç—å —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –Ω–∞—á–∞–ª–∞
- **pop_back()** ‚Äî —É–¥–∞–ª–∏—Ç—å –∏ –≤–µ—Ä–Ω—É—Ç—å —ç–ª–µ–º–µ–Ω—Ç –∏–∑ –∫–æ–Ω—Ü–∞

### –ü—Ä–æ—Å–º–æ—Ç—Ä –±–µ–∑ —É–¥–∞–ª–µ–Ω–∏—è
- **peek_front()** ‚Äî –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å –ø–µ—Ä–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç
- **peek_back()** ‚Äî –ø–æ—Å–º–æ—Ç—Ä–µ—Ç—å –ø–æ—Å–ª–µ–¥–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç

### –°–æ—Å—Ç–æ—è–Ω–∏–µ
- **is_empty()** ‚Äî –ø—É—Å—Ç –ª–∏ deque
- **size() / len(deque)** ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤

–•—Ä–∞–Ω–∏–º:

## push_front(value)

–°–æ–∑–¥–∞—ë–º –Ω–æ–≤—ã–π —É–∑–µ–ª –ø–µ—Ä–µ–¥ `head`.

–°–æ–∑–¥–∞—ë–º –Ω–æ–≤—ã–π —É–∑–µ–ª –ø–æ—Å–ª–µ `tail`.

–ü–æ—Å–ª–µ `push_back(40)`:

–£–¥–∞–ª—è–µ–º —É–∑–µ–ª `head` –∏ —Å–¥–≤–∏–≥–∞–µ–º `head` –Ω–∞ `head.next`.

–°–ª–æ–∂–Ω–æ—Å—Ç—å: **O(1)**

–£–¥–∞–ª—è–µ–º —É–∑–µ–ª `tail` –∏ —Å–¥–≤–∏–≥–∞–µ–º `tail` –Ω–∞ `tail.prev`.

–°–ª–æ–∂–Ω–æ—Å—Ç—å: **O(1)**


## –ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è (–¥–≤—É—Å–≤—è–∑–Ω—ã–π —Å–ø–∏—Å–æ–∫)

* –º–Ω–æ–≥–æ —Å—Å—ã–ª–æ–∫ (`prev/next`) ‚Üí –±–æ–ª—å—à–µ –ø–∞–º—è—Ç–∏, —á–µ–º –º–∞—Å—Å–∏–≤

## –ò–¥–µ—è

–ú—ã —Ö—Ä–∞–Ω–∏–º:

* `_data` ‚Äî –º–∞—Å—Å–∏–≤
* `_front` ‚Äî –∏–Ω–¥–µ–∫—Å –ø–µ—Ä–≤–æ–≥–æ —ç–ª–µ–º–µ–Ω—Ç–∞
* `_size` ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ —ç–ª–µ–º–µ–Ω—Ç–æ–≤

–≠–ª–µ–º–µ–Ω—Ç—ã –ª–µ–∂–∞—Ç –ø–æ–¥—Ä—è–¥:


–ï—Å–ª–∏ –º–∞—Å—Å–∏–≤ –∑–∞–ø–æ–ª–Ω–µ–Ω ‚Äî –¥–µ–ª–∞–µ–º `_resize()` (–∫–æ–ø–∏—Ä—É–µ–º —ç–ª–µ–º–µ–Ω—Ç—ã –≤ –Ω–æ–≤—ã–π –º–∞—Å—Å–∏–≤ –∏ —Å–±—Ä–∞—Å—ã–≤–∞–µ–º `_front` –≤ 0).


## push_back(value)

–î–æ–±–∞–≤–ª—è–µ–º –≤ –∫–æ–Ω–µ—Ü: –∏–Ω–¥–µ–∫—Å –∫–æ–Ω—Ü–∞ —Ä–∞–≤–µ–Ω

–ï—Å–ª–∏ –º–µ—Å—Ç–∞ –Ω–µ—Ç (–∫–æ–Ω–µ—Ü —É–ø–∏—Ä–∞–µ—Ç—Å—è –≤ `len(_data)`), –¥–µ–ª–∞–µ–º resize.

–°–ª–æ–∂–Ω–æ—Å—Ç—å: **–∞–º–æ—Ä—Ç–∏–∑–∏—Ä–æ–≤–∞–Ω–Ω–æ O(1)**


## pop_front()

–£–¥–∞–ª—è–µ–º –ø–µ—Ä–≤—ã–π —ç–ª–µ–º–µ–Ω—Ç:

* –±–µ—Ä—ë–º `_data[_front]`
* `_front += 1`
* `_size -= 1`

–ï—Å–ª–∏ –æ—á–µ—Ä–µ–¥—å —Å—Ç–∞–ª–∞ –ø—É—Å—Ç–æ–π ‚Äî `_front = 0`.

–°–ª–æ–∂–Ω–æ—Å—Ç—å: **O(1)**


## pop_back()

–£–¥–∞–ª—è–µ–º –ø–æ—Å–ª–µ–¥–Ω–∏–π —ç–ª–µ–º–µ–Ω—Ç:

* –∏–Ω–¥–µ–∫—Å –ø–æ—Å–ª–µ–¥–Ω–µ–≥–æ: `_front + _size - 1`
* —É–º–µ–Ω—å—à–∞–µ–º `_size`

–°–ª–æ–∂–Ω–æ—Å—Ç—å: **O(1)**


## peek_front / peek_back

* `peek_front()` ‚Üí `_data[_front]`
* `peek_back()` ‚Üí `_data[_front + _size - 1]`

–°–ª–æ–∂–Ω–æ—Å—Ç—å: **O(1)**


## push_front(value) ‚Äî –≤–∞–∂–Ω—ã–π –º–æ–º–µ–Ω—Ç

–í —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ **–±–µ–∑ –∫–æ–ª—å—Ü–∞** —ç—Ç–æ ‚Äú–Ω–µ—É–¥–æ–±–Ω–∞—è‚Äù –æ–ø–µ—Ä–∞—Ü–∏—è.

–ï—Å—Ç—å 3 —Å–ª—É—á–∞—è:

### 1) deque –ø—É—Å—Ç

–ü—Ä–æ—Å—Ç–æ –∫–ª–∞–¥—ë–º –≤ `data[0]`.

### 2) —Å–ª–µ–≤–∞ –µ—Å—Ç—å –º–µ—Å—Ç–æ (`_front > 0`)

–°–¥–≤–∏–≥–∞–µ–º `_front` –≤–ª–µ–≤–æ –∏ –∑–∞–ø–∏—Å—ã–≤–∞–µ–º:

```

```

–°–ª–æ–∂–Ω–æ—Å—Ç—å: **O(1)**

### 3) —Å–ª–µ–≤–∞ –º–µ—Å—Ç–∞ –Ω–µ—Ç (`_front == 0`)

–¢–æ–≥–¥–∞ –Ω—É–∂–Ω–æ ‚Äú–æ—Å–≤–æ–±–æ–¥–∏—Ç—å –º–µ—Å—Ç–æ‚Äù —Å–ª–µ–≤–∞.

–ï—Å–ª–∏ –º–∞—Å—Å–∏–≤ –ø–æ–ª–æ–Ω ‚Äî —Ä–∞—Å—à–∏—Ä—è–µ–º.
–ü–æ—Å–ª–µ —ç—Ç–æ–≥–æ –¥–µ–ª–∞–µ–º **—Å–¥–≤–∏–≥ –≤—Å–µ—Ö —ç–ª–µ–º–µ–Ω—Ç–æ–≤ –≤–ø—Ä–∞–≤–æ –Ω–∞ 1**:

```

```

–≠—Ç–æ —Å—Ç–æ–∏—Ç **O(n)**.

üëâ –ò–º–µ–Ω–Ω–æ –ø–æ—ç—Ç–æ–º—É ‚Äúdeque –±–µ–∑ –∫–æ–ª—å—Ü–∞‚Äù ‚Äî —É—á–µ–±–Ω—ã–π –≤–∞—Ä–∏–∞–Ω—Ç.
–í —Ä–µ–∞–ª—å–Ω—ã—Ö —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è—Ö deque –æ–±—ã—á–Ω–æ –¥–µ–ª–∞—é—Ç **–Ω–∞ –∫–æ–ª—å—Ü–µ–≤–æ–º –±—É—Ñ–µ—Ä–µ**, —á—Ç–æ–±—ã `push_front` –±—ã–ª O(1).


## –ü—Ä–∏–º–µ—Ä –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è (–º–∞—Å—Å–∏–≤, –±–µ–∑ –∫–æ–ª—å—Ü–∞)

**–ü–ª—é—Å—ã:**
