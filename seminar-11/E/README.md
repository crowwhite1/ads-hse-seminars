# E. Дима и ставки

## Описание задачи

Дан массив чисел, где каждое число описывает результат ставки за день:

- положительное число — выигрыш;
- отрицательное — проигрыш.

Нужно найти максимально длинную подпоследовательность дней, в которой каждый следующий результат строго больше предыдущего:

- выигрыш становится больше;
- проигрыш становится меньше по модулю, то есть число тоже становится больше.

Также требуется вывести саму найденную подпоследовательность.

## Идея решения

Это задача на **наибольшую возрастающую подпоследовательность**.

Обозначим:

- `dp[i]` — длина наибольшей возрастающей подпоследовательности, заканчивающейся в позиции `i`;
- `prev[i]` — индекс предыдущего элемента в восстановленном ответе.

Тогда для каждого `i` перебираем все `j < i`:

- если `a[j] < a[i]`, то элемент `a[i]` можно поставить после `a[j]`;
- если `dp[j] + 1 > dp[i]`, обновляем `dp[i]` и `prev[i]`.

## Почему это работает

Любая возрастающая подпоследовательность, заканчивающаяся в `i`, должна перед этим заканчиваться в некоторой позиции `j < i`, где `a[j] < a[i]`.

Если выбрать среди всех таких `j` тот, который даёт максимальное `dp[j]`, то получим оптимальную подпоследовательность, заканчивающуюся в `i`.

После вычисления всех `dp[i]` остаётся выбрать позицию с максимальным значением `dp`.

## Восстановление ответа

Когда найден индекс `pos`, в котором заканчивается лучший ответ:

- добавляем `a[pos]` в результат;
- переходим в `prev[pos]`;
- повторяем, пока индекс не станет `-1`.

Так подпоследовательность восстанавливается с конца, поэтому в конце её нужно развернуть.

## Алгоритм

1. Считать массив.
2. Инициализировать `dp[i] = 1`, `prev[i] = -1`.
3. Для каждого `i` перебрать все `j < i`.
4. Если `a[j] < a[i]`, попробовать улучшить `dp[i]`.
5. Найти позицию с максимальным `dp[i]`.
6. Восстановить подпоследовательность через массив `prev`.

## Сложность

Для массива длины `n`:

- время: `O(n^2)`;
- память: `O(n)`.

## Итог

Задача решается классической динамикой для наибольшей возрастающей подпоследовательности с восстановлением ответа.
