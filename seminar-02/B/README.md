# Задача B — Сортировка простыми вставками

## Условие
Необходимо отсортировать массив по неубыванию с помощью сортировки
простыми вставками и посчитать количество элементов, которые при добавлении
в отсортированную часть **уже находились на своём месте**, то есть их не пришлось сдвигать.

## Идея решения
Используется классический алгоритм insertion sort:

- элементы массива по одному добавляются в уже отсортированную левую часть
- для каждого элемента проверяем, пришлось ли его сдвигать влево
- если элемент сразу оказался на нужной позиции, увеличиваем счётчик

Таким образом считаются элементы, которые **не требовали перемещения**.

## Сложность
- Время: O(n²) в худшем случае
- Память: O(1) (не считая массива)
