# Задача C — Метод вставками

## Условие
Необходимо отсортировать массив по возрастанию методом сортировки вставками.
После каждого шага **вставки**, при котором элементы массива действительно
сдвигались, требуется вывести текущее состояние массива.

Если на шаге вставки элемент сразу оказался на своём месте,
вывод для этого шага не производится.

## Идея решения
Используется классический алгоритм insertion sort:

- массив постепенно расширяется слева направо
- каждый новый элемент вставляется в отсортированную часть
- если при вставке происходили сдвиги элементов, выводится текущее состояние массива

Таким образом выводятся только те шаги, где массив реально изменился.

## Сложность
- Время: O(n²) в худшем случае
- Память: O(1) (не считая массива)
