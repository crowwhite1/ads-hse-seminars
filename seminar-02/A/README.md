# Задача A — Сортировка пузырьком

## Условие
Нужно отсортировать массив по неубыванию с помощью сортировки пузырьком
и посчитать количество обменов (swap), совершённых при сортировке.

## Идея решения
Используется классический bubble sort:

- на каждом проходе «всплывает» максимальное значение на правый край
- если за проход не было ни одного обмена, массив уже отсортирован — можно завершать раньше

Количество обменов увеличивается каждый раз, когда меняем местами соседние элементы.

## Сложность
- Время: O(n^2) в худшем случае, O(n) в лучшем (если почти отсортировано и сработает ранний выход)
- Память: O(1) (не считая массива)
