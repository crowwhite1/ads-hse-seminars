# Задача D — Успеваемость

## Идея решения
Пусть у Вовочки:
- `a` двоек, `b` троек, `c` четвёрок
- добавим `d` пятёрок

Итоговая оценка — среднее арифметическое, округлённое до ближайшего целого.
Чтобы округление дало не меньше 4, достаточно, чтобы среднее было не меньше `3.5`.

Требование:
(2*a + 3*b + 4*c + 5*d) / (a + b + c + d) >= 3.5

Домножаем на 2 и упрощаем:
3*d >= 3*a + b - c

Отсюда:
- если `3*a + b - c <= 0`, то `d = 0`
- иначе `d = ceil((3*a + b - c) / 3)`

## Сложность
- Время: O(1)
- Память: O(1)
