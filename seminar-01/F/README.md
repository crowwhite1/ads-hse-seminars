# Задача F — k-е простое число

## Условие
По данному натуральному числу `k` (`k ≤ 100000`) вывести `k`-е по счёту простое число.
Использовать массив для запоминания уже найденных простых чисел.

## Идея решения
Используется решето Эратосфена:
1. Создаём булев массив `is_prime`, где `is_prime[x] = True`, если `x` простое.
2. Просеиваем числа до `limit`, вычёркивая кратные.
3. Проходим по массиву и считаем простые числа, пока не найдём `k`-е.

Верхняя граница `limit = 2_000_000` выбрана с запасом и гарантированно покрывает `k ≤ 100000`.

## Сложность
- Время: O(n*log(log(n)))
- Память: O(n)
