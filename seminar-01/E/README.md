# Задача E — Разложение на простые множители

## Условие
Дано целое число `N` (`2 ≤ N ≤ 10^9`).
Необходимо разложить его на простые множители
и вывести результат в порядке возрастания.

## Идея решения
Разложение выполняется перебором возможных делителей
от `2` до `sqrt(N)`.

Для каждого делителя подсчитывается, сколько раз он входит в `N`.
Если делитель встречается один раз, он выводится без степени,
иначе — в виде `p^k`.

Если после перебора осталось число больше 1,
оно является простым множителем и добавляется в результат.

## Сложность
- Время: O(√N)
- Память: O(1) (не считая вывода)
