# Задача G — Разделим все!

## Условие
Даны `n` целых чисел.
Необходимо найти максимальное целое число `k`,
на которое делятся все данные числа.

## Идея решения
Искомое число — это **наибольший общий делитель (НОД)**
всех чисел.

НОД для нескольких чисел можно найти последовательно:
gcd(a₁, a₂, a₃, ...) = gcd(gcd(a₁, a₂), a₃, ...)

Для вычисления используется алгоритм Евклида.

## Реализации
В задаче представлены два подхода:
1. Использование встроенной функции `gcd` (где доступно).
2. Самописная реализация алгоритма Евклида.

## Сложность
- Время: O(n · log(max(aᵢ)))
- Память: O(1)
