# Задача C — Проверка на простоту

## Условие
Дано натуральное число `n` (`n ≤ 2·10^9`, `n ≠ 1`).
Необходимо определить, является ли число простым.

## Идея решения
Число является простым, если у него нет делителей,
кроме 1 и самого себя.

Достаточно проверять делители от `2` до `sqrt(n)`.

## Реализации

В задаче представлены два подхода:

1. **Через логический флаг**  
   Если найден делитель — число составное.

2. **Аналог `for … else` (Python)**  
   В Java и C++ используется дополнительный флаг,
   который позволяет определить, был ли выполнен `break`.

## Сложность
- Время: O(√n)
- Память: O(1)
