# Задача B — Числа Фибоначчи

## Идея решения
Нужно вывести `n`-е число Фибоначчи при определении:
- `F(0) = 0`
- `F(1) = 1`
- `F(n) = F(n-1) + F(n-2)`

## Реализации
В папке представлены три подхода:

1. **Итеративно (2 переменные)**  
   Поддерживаем два последних значения и обновляем их в цикле.  
   Время: O(n), память: O(1).

2. **Рекурсия с мемоизацией**  
   Используем кеш (memo / lru_cache), иначе рекурсивное решение слишком медленное.  
   Время: O(n), память: O(n).

3. **Через массив/список**  
   Заполняем массив `fib[0..n]`.  
   Не оптимально по памяти, так как хранится весь массив значений.  
   Время: O(n), память: O(n).
